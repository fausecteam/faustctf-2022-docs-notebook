Home for all your great research powered by the wonderful Org Syntax!

Brief Syntax Introduction

1 Preamble

You can set the title of your document with the #+TITLE: directive and the author via #+AUTHOR:.

Furthermore, you can change export settings with #+OPTIONS:

These options may be useful:

Option Description Example
toc Generate a Table of Contents toc:nil
num How many headlines to number num:2

For a full list, see: Export Settings

Example: 

		  1: #+TITLE: How to time travel
		  2: #+AUTHOR: Doc Brown
		  3: #+DATE: 1955-11-05

2 Tables

A table cell is delimited by |. A horizontal line in the table is given by |---------|

2.1 Table Formulas

Your Table may include a #+TBLFM: line at the end of the table.

On this line you can supply multiple table formulas, seperated by ::

These table formulas will be evaluated on export.

Example: 

			  1: |      Item | Price | Amount | Total |
			  2: |-----------+-------+--------+-------|
			  3: | Plutonium |   100 |      5 |       |
			  4: |  Delorean | 80000 |      1 |       |
			  5: |-----------+-------+--------+-------|
			  6: |           |       |        |       |
			  7: #+TBLFM: $4=$2*$3::@>$4=vsum(@2$4..@-1$4)

exports to:

Item Price Amount Total
Plutonium 100 5 500
Delorean 80000 1 80000
      80500

3 Source code blocks

Source code blocks are of the following form: 

			1: #+begin_src emacs-lisp
			2: (defun foo (x)
			3:   (format "baz: %s" x))
			4: 
			5: (foo "bar")
			6: #+end_src

However, code block evaluation is disabled due to security concerns.

Take a look at the manual to see all features.
